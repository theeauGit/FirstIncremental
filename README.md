# PenFactory 
Petit projet personnel pour créer un jeu incrémental en JavaScript et explorer les concepts du développement front-end.

## Objectif du projet

L’idée derrière ce projet est simple :

Apprendre à gérer les grands nombres et les progressions exponentielles typiques des jeux incrémentaux.

Pratiquer les bases de JavaScript, notamment la manipulation du DOM et la gestion d’événements.

Découvrir le développement front-end en construisant une interface interactive et dynamique.

En créant ce jeu, je peux expérimenter les boucles de progression, les systèmes de ressources et les mechanics d’incrémentation dans un contexte concret, tout en consolidant mes compétences en JS et front-end.

## Langages utilisées

JavaScript : logique du jeu, calculs, progression des ressources.

HTML / CSS : structure et style de l’interface utilisateur.

## Fonctionnalités principales

Clics et gains progressifs.

Gestion de ressources en nombre croissant (big numbers).

Améliorations et upgrades simples pour accélérer la progression.

Interface front-end minimaliste pour visualiser l’avancement.

## Pourquoi ce projet ?

Je voulais un projet concret et motivant pour pratiquer JS et le front-end, tout en ayant un résultat visible et interactif. Les jeux incrémentaux offrent un excellent terrain pour comprendre la progression numérique, expérimenter la logique conditionnelle et tester les interactions utilisateurs.
